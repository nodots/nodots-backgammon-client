# Nodots Backgammon

Nodots Backgammon is a React implementation of the oldest boardgame in the world.

## Goals

- Integrate with any Backgammon API that is RESTful
- Auditable dice rolls
  - User level
  - Game level
  - Player level
  - Tournament level
  - Forever level
